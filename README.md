# ParaTranz Undertale Previewer

ParaTranz で翻訳中のテキストを、Undertale 形式のメッセージボックスでリアルタイムにプレビューするための Chrome 拡張機能です。

## 導入方法

1.  Google Chrome の設定から **「拡張機能」** をクリック（またはアドレスバーに `chrome://extensions/` と入力）
2.  右上にある **「デベロッパーモード」** を有効にする
3.  **「パッケージ化されていない拡張機能を読み込む」** をクリック
4.  このマークダウンがあるフォルダを選択

## ショートカットキー

- `Alt + U`: プレビューの表示 / 非表示を切り替え
- `Alt + P`: タイピングアニメーションを再生
